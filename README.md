# Mastering-Linked-Lists 🚀

A comprehensive and well-structured  repository dedicated to the implementation and mastery of **Linked Lists**. This project covers everything from basic node manipulation to advanced operations in both Singly and Doubly Linked Lists, along with practical applications of the Standard Template Library (STL). 

---
 
## 📂 Repository Contents 

### 1. Singly Linked List 
Detailed implementations covering fundamental operations:
* **Insertion:** At Beginning, At End, and Insert After a specific node.
* **Deletion:** Delete First Node, Delete Last Node, and Delete a specific Node.
* **Searching:** Find Node and General Search operations.
* **Core Logic:** `Singly Linked List Implementation` folder.

### 2. Doubly Linked List
Advanced implementations handling both `Next` and `Previous` pointers:
* **Insertion:** (At Beginning, At End, Insert After).
* **Deletion:** (Delete First Node, Delete Last Node, Delete Node).
* **Core Logic:** `Doubly Linked List Implementation` folder.

### 3. Additional Data Structures & STL
* **Stack & Queue:** Practical usage of the Standard Template Library (`STL-Stack` and `STL-Queue`).
* **Stack Swap:** A specific logic exercise for swapping stack contents.
* **Map:** Implementation and usage of the `Map` data structure.

## 🚀 How to Use
1. **Clone the repository:**

```bash
git clone [https://github.com/YOUR_USERNAME/Linked-List-Practice.git](https://github.com/YOUR_USERNAME/Mastering-Linked-Lists-CPP.git)
```

2. **Navigate to a specific folder:**

```bash
cd "Operations - Insert At End"
```

3. **Compile and Run:**
Using a compiler like G++:
```bash
g++ main.cpp -o app
./app
```
